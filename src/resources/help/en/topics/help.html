Introduction

The PDFIndexer is a pure JavaTM application, which makes document-indexes searching PDF (Printable Document Format) documents.

Usage

With PDFIndexer we can view PDF (Printable Document Format) documents and make document-indexes.

Document viewing

Open

After the application start there is not opened any document. To view a document we have to open one. We can do that in two ways:

  • From the menu we choose File > Open > File....
  • In the toolbar we click on the Open button.

In the upcomming dialogbox we choose the needed document, an the application will open that. At the left side of the application window will apear the navigation and the indexer panels, at the left side the first page of the document, in the toolbar the navigation and the viewing buttons will be enabled.

Comment: If a document is open and we want to open another document, the application will close automatically the opened document.

Close

We can close the opened document choosing File > Close from the menu.

Save a copy

We can make a copy of an opened document:

  • Choosing File > Save a copy from the menu.
  • Clicking Save a copy in the toolbar.

In the upcomming dialogbox we hava to give a file name and press Save a copy.

Navigation

The application shows yust one page of the opened document. To view the document, we can use the navigation panels and the navigation buttons.

Navigation panels

The navigation panels are positioned at the left side of the application, and one contains the bookmarks, the other the pages.

Clicking on the bookmarks (if there are), in the right panel will appear the page with the paragraph pointed by the bookmark.

Clicking on the pages, will appear the begining of the chosen page.

Navigation buttons

With the navigation buttons from the toolbar we can navigate to the First page, Previous page, Next page, Last page, or typing a number to the given page.

The same actions can be done with the subitems of View > Go menuitem.

View

The view of the document can be changed with the rotation and zooming of the current page view. This can be done with the Zoom and the Rotate buttons from the toolbar, as well as with the corresponding menuitems from the View menu.

Comment: The percentage of the zoom is not yust choosable, rather it could be given arbitrarily.

Indexer

To make document-indexes is the main point of the PDFIndexer application. This can be done by clicking on the Indexer tab in the left-side panel.

Before search

Before search we have to give the entries an the search conditions.

Entries

The entries can be given by typing in the text-editor located at the bottom of the panel, or by pressing the Import button and loading an external file. If a special character can't be typed, we can add by pressing the Insert button. By definition, entry is any separate row in the text-editor: word or expression. Hence, the imported bunch of entries have to conform to this rule.

Search conditions

Search conditions can be:

  • The case matching.
  • The word matching.
  • The strict character matching. If is not checked, the similar accentuated characters are handled just the same. I.e.: the character û is just the same as the character ű.
  • The regular expression matching.
  • The finding all matches on a page.
  • The search-intervals.

With the search-intervals we can split the document in several parts. Also in this way it is possible to give the start-number of paging an to define the look of the index pages: with prefix, with suffix, in roman numerals and in upper or lower case.

Comment: The page numbering in the document-index restarts from the first page of any given interval.

The search can be run pressing the Start button.

During the search

During the search we can see the progress of the action, we can stop the search by pressing the Stop button and, if there are partial results, we can see their location in the right panel by clicking on the page numbers.

After search

After search (whether the search has completed or was stopped) we can save the resulting document-index with the Export button, or we can define a new search after pressing the New button.
